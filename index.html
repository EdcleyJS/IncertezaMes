Média Hipotética.

  • O objetivo dessa técnica é exibir uma amostra extraida de resultados hipotéticos (sorteios) da distribuição de médias. Para cada área, é criado um mapa que se torne uma espécie de quadro em uma apresentação animada. Você terá que juntar informações de vários quadros, usando o sistema visual ou algum processo mais mecânico, como a contagem para entender os dados.A média da distribuição representada pela área é mapeada como uma cor para a área correspondente. Essa visualização usa um esquema de cores de extremidade única; quão alta ou baixa é a média de cada área em cada quadro.

  • Quando estiver pronto para continuar click em Avançar.

Média Hipotética.

  • Nessa tela você verá a média mudando ao longo de meses aleatórios. E você deverá tomar decisões observando o comportamento das áreas.
  • O Ponto 1 indica o nome da técnica que é exibida.
  • O ponto 2 mostra que ao passar o mouse por cima das áreas detalhes são exibidos.

  • O ponto 5 contém informações referente aos dados mostrados no mapa destacando um mês.

Visualização 01 - Comparação com um valor limite.

  • O Objetivo desta visualização é permitir que você compare um determinado valor às distribuições de probabilidade acumulada (% - chances) representadas pelas áreas do mapa. Você pode arrastar uma linha horizontal que representa um valor constante;
    a probabilidade de a distribuição ser maior que esse valor constante é mapeada como uma cor para a área correspondente.
    Uma escala de cores divergente é usada para representar o espaço total de "totalmente baixo" a "totalmente alto".

Visualização 02 - Comparação com intervalo.

  • A visualização de Intervalo é semelhante à comparação com uma constante. Ela é usada para avaliar a probabilidade do valor de uma distribuição cair dentro de um intervalo. Você pode arrastar e dimensionar uma faixa horizontal. A probabilidade (% - chance) de a distribuição representada pela área estar contida na região é mapeada como uma cor para a área correspondente. Essa visualização usa um esquema de cores de extremidade única; se é provável que o valor esteja dentro ou fora do intervalo.

Visualização 03 - Pontos de Densidade.

  • Pontos de distribuição consiste em desenhar pontos que representam valores de uma distribuição para uma dada área. Ela é usada para avaliar a probabilidade usando o sistema visual ou algum processo mais mecânico, como a contagem dos pontos para entender os dados. A distribuição representada pelos pontos nas áreas é mapeada como uma cor para o valor do ponto correspondente. Essa visualização tambémusa um esquema de cores de extremidade única; se é o valor do ponto é relativamente alto ou baixo.

  • Quando estiver pronto para continuar click em Avançar.
