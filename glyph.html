Overview

  • Esta visualização exibe glyphs para representar a média e o desvio para cada área.

    Quanto maior a média o ponto se desloca para cima.

    Quanto maior o desvio o ponto se desloca para direita.

    É possível utilizar filtros para refinar os valores.

    Ao desabilitar os filtros o valor padrão é restaurado.
